GET /ranking

Tags: ranking, data

Retrieve rankings for users that participate in the experiments. Rankings can be limited by date and experiment id. Date can be provided in two formats: "yyyy-MM-dd" and "yyyy-MM-dd'T'HH:mm".
after

the time to limit user contribution rankings by.

 query string
experimentId

the id of an existing experiment.

 query string

Uses default content-types: application/json

200 OK

A sorted array of ranked users.

Body Schema: object[]

Reference: RankedUserData
