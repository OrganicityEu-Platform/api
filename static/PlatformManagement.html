• Overview

OrganiCity Platform Management API

Connect new sites to the OrganiCity platform, as well as perform the management of its related attributes

Schemes: https

Summary

Path Operation Description
/sites GET

retrieve the list of sites.
POST

register a new OrganiCity site
/sites/{siteId} DELETE

retrieve the list of services for a given site.
POST

register a new OrganiCity service for a given site
/sites/{siteId}/services/{serviceId} DELETE

deletes a service
GET

find service by ID
POST

updates a service in the facility
